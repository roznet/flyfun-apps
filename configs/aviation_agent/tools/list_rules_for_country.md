List aviation rules and regulations for a specific country (iso-2 code eg FR,GB), including customs, flight plans, and operational requirements.

**Parameters:**
- country_code: ISO-2 country code (e.g., FR, GB, DE)
- tags: Optional array of topic tags to filter (e.g., ["flight_plan", "transponder", "airspace"])

Can be filtered by tags like flight_plan, transponder, airspace, permission, procedure, etc.
